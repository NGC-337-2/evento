Base URL: http://localhost:5000/api/v1

## 🔐 Authentication
Note: Most routes require a JWT token in the Authorization header.
Header: Authorization: Bearer <TOKEN>

## 📅 Events

### 4. Create Event
POST /events
Type: multipart/form-data
Body:
- title: "Tech Conference 2026"
- description: "Networking and talks"
- category: "technology"
- date: "2026-10-10T10:00:00Z"
- location[venue]: "Grand Hall"
- location[city]: "New York"
- capacity: 200
- price: 50
- image: (File upload)

### 5. Get All Events (Public)
GET /events
Query Params: ?category=technology&search=tech&page=1

### 6. Get Featured Events
GET /events/featured

### 7. Admin: Approve Event
PUT /events/:id
Body (JSON):
{
    "status": "published"
}
